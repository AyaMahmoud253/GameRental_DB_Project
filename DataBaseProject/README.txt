<h1>Game Rental</h1>
<p>This project stores data base for a game rental store and performs several other works such as:
1- He prints the best vendor who was able to rent the largest possible number of games
2- He prints the most renter who has rented the most games
3- He prints the games that were not rented
4- It prints the most rented games
5- It prints the worst vendor who could not rent games
6- It prints the vendor that has not added any game to the store</p>

<h3>Note: You can add your database and try queries.</h3>


The C# code implements queries via buttons linked to the sql server management.
